<h3>课程介绍：</h3>
<p>升级一：技术栈“除旧迎新”。</p>
<p>剔除了JSP、EL、JSTL等过时技术，新增了企业端流行技术Thymeleaf，新增了热门的开发技术Vue和Axios框架，课程设计上不仅贴合企业实际生产环境，同时考虑学习者的技术成长，新增了自定义SSM框架的实现。</p>
